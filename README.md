# Study Buddy AI 🤖📚

AI-powered study assistant that helps students manage subjects, generate quizzes, track progress, and stay productive with smart analytics.

## 🚀 Features
- 📊 **Learning Dashboard** – Track XP, streaks, and learning progress.
- 🧠 **AI Mentor** – Get guidance and explanations for study topics.
- 📝 **Quiz Generator** – Create quizzes automatically from subjects.
- ⏱ **Focus Timer** – Improve productivity using timed study sessions.
- 📚 **Subject Manager** – Organize subjects and track completion.
- 📈 **Telemetry Analytics** – Analyze learning patterns and performance.

## 🛠 Tech Stack
- **Frontend:** TypeScript / Web UI  
- **Backend:** Node.js  
- **Database:** PostgreSQL  
- **Deployment:** Replit  
- **Package Manager:** PNPM
